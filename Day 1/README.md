Topic: Print Statement
Summary: Print function shows any message you want to display on the console.
The print function uses parenthesis () and quotation mark for strings.

Example: print("This is my first day of learning python")

Topic: Variables
Summary: Variables are used to store values that you can use multiple times.
You can also change the value of a variable by reassigning a new value to it.
Use equal sign (=) to assign a value to a variable.

Examples:
name = "Chan"
x = 3
y = 2.8
is_student = False

Topic: Data Types
Summary: There are multiple data types in programming language.
The common data types are strings, integer, float, boolean, etc...
To know more about data types, you may visit this website: https://www.w3schools.com/python/python_datatypes.asp

Topic: Checking data types
Summary: Python's type() function is used to help checking the data type of information stored in a variable.
To know how to check the data type of the variable name, you may use the print function and type function in one line of code.
Example: name = "Chan"
print(type(name))
output: <class 'str'>
