Summary: Algorithm is a set of instructions that solve a problem while Data structures hold and manipulate data when we execute an algorithm.

1. Linked Lists - is a sequence of data connected through links. Each element is called a node and has two parts. The first part is the data and the second part is a pointer to the next node. The first node is the head, and the final node is the tail. This data doesn't need to be stored in contiguous blocks of memory, so it can be located in any available memory address. Linked lists can be used to implement other data structures like stacks, queues, and graphs. A common application of a linked list is to access information by navigating backward and forward, such as on a web browser or music playlist.
